# Kidooo AI — Official Website

The marketing website for **Kidooo AI**, an AI copilot for autism clinics and families.

Live site: [kellyaicad.github.io/kidoooweb](https://kellyaicad.github.io/kidoooweb)
Product app: [family.kidooo.ai](https://family.kidooo.ai)

## About Kidooo AI

Kidooo AI is a multimodal AI agent for personalized autism care. We transform therapy sessions and home videos into clinician-ready documentation for therapists and actionable guidance for families.
